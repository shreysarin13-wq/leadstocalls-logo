FAQ

The common questions UK sauna businesses ask before launching ads + automation.

GDPR note

We implement consent language and data handling best practices. You remain the data controller; tooling can be configured to your compliance requirements.

Do you only work with garden saunas?
This offer is designed for garden saunas (builders/retailers/installers) in the UK. If you’re adjacent (hot tubs, outdoor living), the system still works — but the copy/offer changes.
What does “Synthflow bot” do?
It can answer after-hours calls/messages, ask qualifying questions, send links, and help with booking. It logs transcripts and tags so you can follow up properly.
What does the human setter do?
During office hours, a human confirms details, handles objections, and gets qualified prospects to book (and show up). Your sales team gets cleaner calendars.
Do I need a CRM?
Not required. We can start with email + Google Sheets, then upgrade to a CRM when volume makes it worth it.
How fast can this go live?
If you have basic assets (logo, photos, offers), a first version can typically go live in days — then we iterate weekly.
